Puheenvuorosovellus


Ominaisuudet:
Kokoushuoneet (Meetings)
 - Käyttäjät voivat luoda kokoushuoneita
    - Kokoushuoneilla on nimi ja valinnainen salasana
    - Lisäksi kokoushuoneille voi määrittää asetuksia (TODO)
    - Käyttäjät voivat liittyä kokoushuoneisiin

Puheenvuorot (Speeches)
 - Käyttäjät voivat pyytää puheenvuoroa kokoushuoneessa
 - Puheenvuorot asetetaan jonoon
 - Puheenvuoroja on erilaisia
 - Kokouksen puheenjohtaja tai sihteeri voi myöntää puheenvuoron
 - Käyttäjät näkevät puheenvuorojonon tilan reaaliajassa ja nykyisen puheenvuoron

Esitykset (Proposals)
 - Käyttäjät voivat tehdä esityksiä kokoushuoneessa
 - Käyttäjät voivat kannattaa esityksiä
 - Käyttäjät voivat muokata omia esityksiään
 - Kokouksen puheenjohtaja tai sihteeri voi käsitellä esityksiä

Äänestykset (Votes)
 - Puheenjohtaja tai sihteeri voi aloittaa äänestyksen yhdestä tai useammasta esityksestä
 - Äänestykseen odotetaan kaikkien kokoushuoneessa olevien käyttäjien ääniä
 - Äänestys voi olla avoin tai suljettu
 - Käyttäjät voivat äänestää esityksistä kokoushuoneessa
 - Äänestystulokset näytetään reaaliajassa
